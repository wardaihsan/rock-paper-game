# rock-paper-game
I have created a simple rock paper game  using python.
This game is based on simple if and else statements .
I first created a variable to enter users choice such as rock paper scissors which is then compared to options shown on screen which are 1.rock 2.paper 3.scissors in case of any invalid choice the program prints invalid statement.
Then I created a variable named comp_choice and imported random file so the computer chose randomly from the given array.
In the next step I created if statement to check if both the choices of the computer and the user are same and printed match tied.
In the next step i created elif statement for when the user entered rock and created conditional statements.
second elif statement was created for when the user entered paper.
third elif statement was created for when the user entered scissors.
Then the winner of the game is shown on then on the screen .
